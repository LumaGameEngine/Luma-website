The LOVE Language

Lightweight, sandboxed, and designed for game logic. LOVE runs on the Luma Engine VM with a runtime loop at 60 FPS.

Introduction

LOVE (Luma Object & Virtual-machine Environment) is a lightweight, sandboxed scripting language designed for the Luma Engine. It compiles to bytecode and runs on the Luma VM at 60 FPS.

Variables

Control Flow

Loop

// Infinite game loop
loop
    color 20 20 40
    clear
    // Game logic here
    present
end

Conditional Jump

get x
push_int 100
gt // x > 100 ?
jump_if label // Jump if true

Input

Keyboard

key_down "w" // Push true if W is pressed
key_down "space"

Mouse

mouse_x // Push mouse X position
mouse_y // Push mouse Y position
mouse_clicked // Push true if clicked

Rendering

Primitives

color 255 0 0 // Set color
clear // Clear screen
fill_circle 433 285 25 // Draw circle
fill_rect 100 100 50 50 // Draw rectangle
present // Show frame

Examples

Bouncing Ball

let x 433
let y 285
let dx 3
let dy 2
let radius 25

loop
    color 20 20 40
    clear
    
    // Move
    get x
    get dx
    add
    set x
    
    get y
    get dy
    add
    set y
    
    color 74 255 0
    get x
    get y
    get radius
    fill_circle
    
    present
end
