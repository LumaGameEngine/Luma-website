Introduction

LOVE (Luma Object & Virtual-machine Environment) is a lightweight, sandboxed scripting language designed for the Luma Engine.

Simple

Learn it in minutes, not months

Safe

Sandboxed execution, no engine crashes

Fast

Compiled to bytecode on a lightweight VM

Integrated

Seamless interaction with the Luma Engine

Instructions

Each instruction is written on its own line:

INSTRUCTION operands

Instructions are case-insensitive (PUSH = push = Push).

Instructions

Stack Operations

Instruction Description Example
PUSH Push a value onto the stack PUSH 42
POP Remove the top value from the stack POP
DUP Duplicate the top value DUP
SWAP Swap the top two values SWAP

Arithmetic Operations

Instruction Description Example
ADD Add top two values PUSH 5 PUSH 3 ADD → 8
SUB Subtract top two values PUSH 5 PUSH 3 SUB → 2
MUL Multiply top two values PUSH 5 PUSH 3 MUL → 15
DIV Divide top two values PUSH 6 PUSH 3 DIV → 2

I/O Operations

Instruction Description Example
PRINT Print the top value PUSH "Hello" PRINT
READ Read input from console READ

Control Flow

If Statement

// if condition
PUSH condition
JMP_IF else_label
PUSH "Condition is true"
PRINT
JMP end_label

else_label:
PUSH "Condition is false"
PRINT

end_label:
HALT

Loop

// While loop
PUSH 0 // counter
PUSH 10 // max

loop_start:
DUP // duplicate counter
DUP // duplicate max
LT // counter < max ?
JMP_IF loop_end

// Loop body
PUSH "Count: "
DUP // duplicate counter
ADD
PRINT

// Increment
ADD 1
JMP loop_start

loop_end:
POP
HALT

Functions

Defining Functions

FUNC add_two
    PUSH 2
    ADD
    RETURN
ENDFUNC

// Usage
PUSH 5
CALL add_two
PRINT // Output: 7
HALT
